# PetShop Manager

Protótipo de sistema web para gerenciamento de petshops, desenvolvido com HTML, CSS e JavaScript puro.

## Objetivo

O objetivo do projeto é criar uma aplicação funcional para praticar lógica de programação no front-end, manipulação do DOM, formulários, validações, localStorage e organização de código.

A primeira versão será um protótipo apresentável para petshops reais, com possibilidade de evolução futura para uma aplicação fullstack utilizando Java, Spring Boot, banco de dados e API REST.

## Funcionalidades da primeira versão

- Dashboard inicial
- Cadastro de clientes
- Cadastro de pets
- Cadastro de serviços
- Agendamento de banho e tosa
- Listagem de agendamentos
- Alteração de status do agendamento
- Salvamento dos dados com localStorage

## Tecnologias utilizadas

- CSS3
- JavaScript
- localStorage
